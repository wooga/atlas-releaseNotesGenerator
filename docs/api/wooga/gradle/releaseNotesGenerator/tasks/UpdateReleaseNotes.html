Package: wooga.gradle.releaseNotesGenerator.tasks

[Groovy] Class UpdateReleaseNotes

            • wooga.gradle.releaseNotesGenerator.tasks.UpdateReleaseNotes

            • A gradle task class which allows to update a releaseNotes text file on github. The task is based on wooga.gradle.github.base.tasks.internal.AbstractGithubTask so all properties apply here as well.

              Example: 

               task(updateReleaseNotes, type:wooga.gradle.releaseNotesGenerator.tasks.UpdateReleaseNotes) {
          releaseNotes = file("path/to/release_notes.md")
          commitMessage = "update"
      
 }
